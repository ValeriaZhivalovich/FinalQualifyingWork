# Perplexity Bot

Telegram бот для работы с Perplexity через Chrome расширение.

## 🚀 Автоматический деплой на сервер

```bash
# 1. Настройка
cp .env.example .env
nano .env         # добавить токены
nano deploy.sh    # указать IP сервера

# 2. Запуск деплоя (делает ВСЁ автоматически)
./deploy.sh
```

Скрипт автоматически:
- Установит Docker на сервере
- Создаст и загрузит архив
- Соберет Docker образ  
- Запустит контейнер
- Проверит работоспособность

## 📱 Telegram команды

- `/start` - начать
- Любой текст - получить ответ от Perplexity

## 🛠️ Локальная разработка

```bash
./start.sh        # с GUI
./start.sh server # без GUI
./stop.sh         # остановить
```
